**Main handler — `Deno.serve(async (req) => { ... })`:**

1. **CORS preflight:** If `req.method === 'OPTIONS'`, return `new Response('ok', { headers: corsHeaders })`

2. **JWT Authentication** (copy pattern from send-test-email):
   - Get `Authorization` header
   - Create user client with anon key + auth header
   - Call `userClient.auth.getUser()` — if no user, return 401

3. **Get user's workspace_id:**
   - Query `profiles` table: `adminClient.from('profiles').select('workspace_id').eq('id', user.id).single()`
   - Store as `workspaceId`

4. **Parse request body:**
   - `const { campaign_id } = await req.json()`
   - If no campaign_id, return 400

5. **Create admin client** (service_role for bulk operations):
   ```typescript
   const adminClient = createClient(SUPABASE_URL, SERVICE_ROLE_KEY)
   ```

6. **Load campaign:**
   - `adminClient.from('campaigns').select('*').eq('id', campaign_id).single()`
   - Verify `campaign.workspace_id === workspaceId` (STRIDE T-03-09 mitigation)
   - If mismatch, return 403 `{ error: 'Campaign not found or unauthorized' }`
   - If campaign.status is not 'draft' and not 'scheduled', return 400 `{ error: 'Campaign already sent or in progress' }`

7. **Load active contacts from target list:**
   ```typescript
   const { data: members } = await adminClient
     .from('contact_list_members')
     .select('contact_id, contacts(*)')
     .eq('contact_list_id', campaign.contact_list_id)
   const activeContacts = members
     ?.map((m: any) => m.contacts)
     .filter((c: any) => c && c.status === 'active') ?? []
   ```
   - If activeContacts.length === 0, return 400 `{ error: 'No active contacts in target list' }`

8. **Update campaign status to 'sending':**
   ```typescript
   await adminClient.from('campaigns').update({
     status: 'sending',
     sent_at: new Date().toISOString(),
     total_recipients: activeContacts.length,
   }).eq('id', campaign_id)
   ```

9. **Process in batches of BATCH_SIZE (50):**
   ```typescript
   let totalSent = 0

   for (let i = 0; i < activeContacts.length; i += BATCH_SIZE) {
     const batch = activeContacts.slice(i, i + BATCH_SIZE)
   ```

   For each batch:

   a. **Prepare recipients** — For each contact in batch:
      - Generate `trackingId = crypto.randomUUID()`
      - Personalize body_html: call `personalizeHtml(campaign.body_html, contact)`
      - Personalize subject: call `personalizeText(campaign.subject, contact)`
      - **CRITICAL ORDER: personalize FIRST, then wrap links, then inject pixel** (per Research pitfall #4)
      - Wrap links: call `wrapLinks(personalizedHtml, trackingId, TRACKING_BASE)` — this returns `{ html, linkMap }`
      - Inject pixel: call `injectPixel(wrappedHtml, trackingId, TRACKING_BASE)`
      - Add unsubscribe link at bottom: append `<p style="font-size:12px;color:#666;margin-top:20px;"><a href="${TRACKING_BASE}/unsub/${trackingId}">Unsubscribe</a></p>` before the closing `</body>` (or at end if no `</body>` tag)

   b. **Insert campaign_recipients rows** (before sending, for referential integrity):
      ```typescript
      const recipientRows = batch.map((contact, idx) => ({
        campaign_id: campaign_id,
        contact_id: contact.id,
        tracking_id: preparedEmails[idx].trackingId,
        delivery_status: 'queued',
        link_map: preparedEmails[idx].linkMap,
      }))
      await adminClient.from('campaign_recipients').insert(recipientRows)
      ```

   c. **Build Resend batch payload:**
      ```typescript
      const resendBatch = preparedEmails.map((email) => ({
        from: `${campaign.from_name} <${campaign.from_email}>`,
        to: [email.contact.email],
        subject: email.subject,
        html: email.html,
        reply_to: campaign.reply_to_email || undefined,
      }))
      ```

   d. **Send via Resend batch API:**
      ```typescript
      const resendResponse = await fetch('https://api.resend.com/emails/batch', {
        method: 'POST',
        headers: {
          'Authorization': `Bearer ${RESEND_API_KEY}`,
          'Content-Type': 'application/json',
        },
        body: JSON.stringify(resendBatch),
      })
      ```

   e. **Handle Resend response:**
      - If `resendResponse.status === 429`: Campaign hit rate limit or daily quota.
        - Update campaign: `status: 'paused'`, `total_sent: totalSent`
        - Return `{ ok: false, error: 'Rate limit reached. Campaign paused.', sent: totalSent, total: activeContacts.length }`
      - If `!resendResponse.ok`: Log error, continue to next batch (partial failure)
      - If success: Parse response `{ data: [{ id }, ...] }`
        - **Store resend_email_id** on each campaign_recipient (index-matched):
          ```typescript
          const { data: resendData } = await resendResponse.json()
          if (resendData) {
            for (let j = 0; j < resendData.length; j++) {
              await adminClient.from('campaign_recipients')
                .update({
                  resend_email_id: resendData[j].id,
                  delivery_status: 'sent',
                  sent_at: new Date().toISOString(),
                })
                .eq('tracking_id', preparedEmails[j].trackingId)
            }
          }
          totalSent += batch.length
          ```

   f. **Rate limiting between batches:**
      ```typescript
      if (i + BATCH_SIZE < activeContacts.length) {
        await new Promise((resolve) => setTimeout(resolve, 300))
      }
      ```

10. **Mark campaign as sent:**
    ```typescript
    await adminClient.from('campaigns').update({
      status: 'sent',
      total_sent: totalSent,
    }).eq('id', campaign_id)
    ```

11. **Return success:**
    ```typescript
    return new Response(JSON.stringify({ ok: true, sent: totalSent, total: activeContacts.length }), {
      headers: { ...corsHeaders, 'Content-Type': 'application/json' },
    })
    ```

**Helper Functions (define above main handler):**

```typescript
function personalizeText(text: string, contact: { first_name: string | null; last_name: string | null; company: string | null; email: string }): string {
  return text
    .replace(/\{\{first_name\}\}/g, contact.first_name ?? '')
    .replace(/\{\{last_name\}\}/g, contact.last_name ?? '')
    .replace(/\{\{company\}\}/g, contact.company ?? '')
    .replace(/\{\{email\}\}/g, contact.email)
}

function personalizeHtml(html: string, contact: { first_name: string | null; last_name: string | null; company: string | null; email: string }): string {
  return personalizeText(html, contact)
}

function wrapLinks(html: string, trackingId: string, baseUrl: string): { html: string; linkMap: Record<string, string> } {
  const linkMap: Record<string, string> = {}
  let linkIndex = 0
  const wrappedHtml = html.replace(
    /href="(https?:\/\/[^"]+)"/g,
    (_, originalUrl) => {
      const idx = String(linkIndex++)
      linkMap[idx] = originalUrl
      return `href="${baseUrl}/click/${trackingId}/${idx}"`
    }
  )
  return { html: wrappedHtml, linkMap }
}

function injectPixel(html: string, trackingId: string, baseUrl: string): string {
  const pixel = `<img src="${baseUrl}/pixel/${trackingId}" width="1" height="1" style="display:none" alt="" />`
  if (html.includes('</body>')) {
    return html.replace('</body>', `${pixel}</body>`)
  }
  return html + pixel
}

function addUnsubscribeFooter(html: string, trackingId: string, baseUrl: string): string {
  const footer = `<p style="font-size:12px;color:#999;text-align:center;margin-top:30px;"><a href="${baseUrl}/unsub/${trackingId}" style="color:#999;">Unsubscribe</a></p>`
  if (html.includes('</body>')) {
    return html.replace('</body>', `${footer}</body>`)
  }
  return html + footer
}
```

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Browser -> send-campaign | User request triggers bulk email send — must verify identity and workspace ownership |
| send-campaign -> Resend API | Sends emails to real recipients — API key must stay server-side |
| send-campaign -> campaign_recipients | Creates per-recipient tracking records — must verify campaign belongs to user |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-09 | Elevation of Privilege | send-campaign | mitigate | Verify campaign.workspace_id matches authenticated user's workspace_id before sending; prevents sending other users' campaigns |
| T-03-10 | Spoofing | send-campaign auth | mitigate | JWT verification via supabase.auth.getUser() — same pattern as send-test-email; unauthenticated requests return 401 |

| T-03-12 | Denial of Service | Rate limit exhaustion | mitigate | Batch size capped at 50; 300ms delay between batches; campaign paused on 429 response with error surfaced to user |
| T-03-13 | Tampering | campaign_id injection | mitigate | campaign_id validated against workspace_id; invalid/unauthorized campaign_id returns 403 |
</threat_model>
