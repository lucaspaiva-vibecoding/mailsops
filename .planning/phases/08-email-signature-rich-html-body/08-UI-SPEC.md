## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding between swatch cells |
| sm | 8px | Toolbar button gaps, popover internal padding |
| md | 16px | Card section spacing, form field gaps |
| lg | 24px | Card padding (p-6), section breaks within Settings Workspace tab |
| xl | 32px | Card padding large (p-8), CampaignPreview outer padding |
| 2xl | 48px | Major section breaks (page-level) |
| 3xl | 64px | Page-level spacing |

Exceptions:
- Toolbar buttons: w-8 h-8 (32px square) — fixed touch target for icon-only buttons. Source: `CampaignEditorToolbar.tsx`.
- Toolbar dividers: w-px h-5 (1px × 20px) — not a spacing token, structural separator. Source: `CampaignEditorToolbar.tsx`.
- Color swatch cells: 20px × 20px (w-5 h-5) — compact palette grid inside toolbar popover.

## Interaction States

### Toolbar Color Picker
- Default: color swatch button shows `Palette` icon at 16px; no active highlight unless a color is applied to current selection
- Active (color applied to selection): toolbar button shows `bg-gray-700 text-gray-100` (same as other active toolbar buttons)
- Popover open: 8 color swatches in 4×2 grid; currently active swatch has `ring-2 ring-indigo-400`; others have `ring-1 ring-gray-600`
- Hover: swatch transitions to `ring-indigo-400`
- Click swatch: applies `textStyle({ color })` via TipTap, closes popover, returns focus to editor
- Click outside: closes popover without applying color

### Toolbar Alignment Buttons
- Default: `text-gray-400` (same as inactive ToolbarButton)
- Active: `bg-gray-700 text-gray-100` (same as other active toolbar buttons); active state from `editor.isActive('textAlign', { textAlign: 'left|center|right' })`
- Note: default alignment (left) shows as active only when paragraph has explicit left alignment set; unset paragraphs do not show any alignment as active

### Signature Editor in Settings
- Mount state: editor loads with existing `signature_html` content or empty
- Placeholder visible when editor is empty (Placeholder extension, same as campaign editor)
- Focused state: editor container `focus-within:ring-2 focus-within:ring-indigo-500` (apply to outer container div)
- Dirty state: no explicit dirty indicator — signature saves with all other workspace fields on "Save workspace settings"
